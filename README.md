# Site-test

Um site simples desenvolvido para testar e praticar funcionalidades essenciais das linguagens fundamentais da web: HTML, CSS e JavaScript.

## 🚀 Objetivo

Este projeto serve como um ambiente de experimentação para:
- Estruturação de conteúdo com **HTML**.
- Estilização e layout com **CSS**.
- Manipulação de DOM e lógica interativa com **JavaScript**.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando:

* **HTML**: Estrutura das páginas.
* **CSS**: Estilização (arquivo `estilo.css`).
* **JavaScript**: Comportamentos dinâmicos (arquivo `script.js`).

## 📂 Estrutura do Repositório

* `index.html`: Arquivo principal contendo a estrutura do site.
* `estilo.css`: Folha de estilo para personalização visual.
* `script.js`: Lógica em JavaScript para as funcionalidades testadas.
* `source/`: Pasta destinada a recursos adicionais (imagens, ícones, etc).

## 🔧 Como Executar

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/WilsonJr001/Site-test.git](https://github.com/WilsonJr001/Site-test.git)
    ```
2.  Navegue até a pasta do projeto.
3.  Abra o arquivo `index.html` em qualquer navegador web.
